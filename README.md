# 🌆 Dilli-Vibe  
### AI-Powered Mood-Based Discovery Platform for Urban Exploration

> Dilli-Vibe is a personalized discovery engine that helps users find the perfect places in Delhi based on their **real-time mood, preferences, and context**.

---

## 🧭 Problem  

Urban users face **decision fatigue** when choosing where to go.

Traditional platforms like Maps or Zomato:
- ❌ Require active searching  
- ❌ Show overwhelming options  
- ❌ Don’t understand user mood  

👉 There is no system that answers:  
**“Where should I go right now based on my mood?”**

---
## 💡 Solution  

Dilli-Vibe transforms place discovery into a **mood-first experience**.

Users input:
- Mood 😌  
- Budget 💰  
- Company 👥  

And get:
➡️ Curated, AI-ranked place recommendations instantly  

---

## 🧠 Core Features  

- 🎯 Mood-Based Recommendations  
- 🧠 Smart Scoring Algorithm  
- ❤️ Favorites System  
- 👤 User Profiles & Onboarding  
- 🗺️ Map Integration  
- ⚡ Fast API Responses  

---
